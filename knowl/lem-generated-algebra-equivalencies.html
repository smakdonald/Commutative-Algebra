Lemma 1.18. Generated Algebra Equivalencies.

The following are equivalent: (a) \(\Lambda\) generates \(S\) as an \(R\)-algebra. (b) Every element in \(S\) admits a polynomial expression in \(\Lambda\) with coefficients in \(\phi(R)\)[^1], i.e.
\begin{equation*} S = \left\{\sum_{\mathrm{finite}} \phi(r_{i_1, \ldots, i_n}) \lambda_1^{i_1} \cdots \lambda_n^{i_n} \mid r_l \in R, \lambda_j\in\Lambda, i_j \geqslant 0 \right\}. \end{equation*}
(c) If \(R[X]\) is a polynomial ring on a set of indeterminates \(X\) in bijection with \(\Lambda\text{,}\) then the \(R\)-algebra homomorphism
\begin{equation*} \begin{CD} R[X]^{n}@>\pi>> S\\\\ x_{i}@>>> \l_{i} \end{CD} \end{equation*}
is surjective.
