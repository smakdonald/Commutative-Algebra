For (3), note that \(f,g\in \cI(X)\) and \(r\in R\) implies \(X\subseteq \cZ(f,g)\) implies \(X \subseteq \cZ(rf+g)\) implies \(rf+g \in \cI(X)\text{,}\) so \(\cI(X)\) is an ideal. If \(f^t\in \cI(X)\text{,}\) then \(f(a_1,\dots,a_n)^t=0\) for all \((a_1,\dots,a_n)\in X\text{,}\) so \(f(a_1,\dots,a_n)=0\) for all \((a_1,\dots,a_n)\in X\text{,}\) and \(f\in \cI(X)\text{.}\)
