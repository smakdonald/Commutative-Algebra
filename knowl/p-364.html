Each \(\fm = (x_1-a_1,\dots,x_d-a_d)\) is a maximal ideal satisfying \(R/\fm \cong k\text{.}\) Moreover, these ideals are distinct: if \(x_i-a_i,x_i-a'_i\) are in the same ideal for \(a_i\neq a'_i\text{,}\) then the unit \(a_i-a'_i\) is in the ideal, so it is not proper. Therefore, our map is injective. To see that it is surjective, let \(\fm\) be a maximal ideal with \(R/\fm \cong k\text{.}\) Each class in \(R/\fm\) corresponds to a unique \(a \in k\text{,}\) so in particular each \(x_i\) is in the class of a unique \(a_i \in k\text{.}\) This means that \(x_i - a_i \in \fm\text{,}\) and thus \((x_1-a_1,\dots, x_d-a_d)\subseteq \fm\text{.}\) Since \((x_1-a_i,\dots, x_d-a_i)\) is a maximal ideal, we must have \((x_1-a_1,\dots, x_d-a_d)= \fm\text{.}\)
