Lemma 7.11. Properties of Dimension and Height.

Let \(R\) be a ring. (a) A prime has height zero if and only if it is a minimal prime of \(R\text{.}\) (b) An ideal has height zero if and only if it is contained in a minimal prime of \(R\text{.}\) In particular, in a domain, every nonzero ideal has positive height. (c) \(\dim(R) = \sup \{ \dim(R/P) \mid P \in \Spec(R) \} = \sup\{ \dim(R/P) \mid P \in \Min(R) \}\)[^1]. (d) \(\dim(R) = \sup \{ \ht(P) \mid P \in \Spec(R) \}=\sup\{ \mathrm{height}(Q) \mid Q \in \mSpec(R) \}\text{.}\) (e) If \(I\) is an ideal, then \(\dim(R/I) = \sup \{ n \mid \exists P_0 \subsetneq P_1 \subsetneq \cdots \subsetneq P_n, P_i \in V(I) \}\text{.}\) (f) If \(P\) is prime,
\begin{equation*} \dim(R/P) + \ht(P) \leqslant \dim(R). \end{equation*}
(g) If \(I\) is an ideal,
\begin{equation*} \dim(R/I) + \ht(I) \leqslant \dim(R). \end{equation*}
(h) If \(W\) is a multiplicative set, then \(\dim(W^{-1}R) \leqslant \dim(R)\)[^2]. (i) If \(P\) is prime, then \(\ht(P)=\dim(R_P)\text{.}\) (j) If \(P \subseteq Q\) are primes, then \(\dim(R_Q/ P R_Q)\) is the supremum of the lengths of saturated chains of primes in \(R\) of the form \(P = P_0 \subsetneq P_1 \subsetneq \cdots \subsetneq P_n = Q\text{.}\)
