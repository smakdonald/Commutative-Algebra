The implication \((\implies)\) is clear. If \(m_1,\dots, m_s \in M\) are such that \(\overline{m_1},\dots,\overline{m_s}\) generate \(M/\fm M\text{,}\) consider \(N= R m_1 + \dots + R m_s \subseteq M\text{.}\) Since \(M/\fm M\) is generated by the image of \(N,\) we have \(M= N + \fm M\text{.}\) By [provisional cross-reference: cite] [[Mathematics/Commutative Algebra/Results/Theorem - NAK 2|NAK2]], \(M=N\text{.}\)
