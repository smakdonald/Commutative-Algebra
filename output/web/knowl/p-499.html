As a consequence of the previous [provisional cross-reference: cite] [[Mathematics/Commutative Algebra/Results/Lemma - Zero in Localization|Lemma]], it follows that if \(R\) is a domain, then the canonical map \(R \to W^{-1}R\) is always injective for any multiplicatively closed set \(W\text{,}\) since every nonzero \(r \in R\) has \(\ann(r) = 0\text{.}\)
