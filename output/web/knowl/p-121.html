The sequence
\begin{equation*} \begin{CD} 0@>>>M@>f>>N \end{CD} \end{equation*}
is exact if and only if \(f\) is injective. Similarly,
\begin{equation*} \begin{CD} M@>f>>N@>>>0 \end{CD} \end{equation*}
is exact if and only if \(f\) is surjective. As a consequence, we see that
\begin{equation*} \begin{CD} 0@>>>M@>f>>N@>>>0 \end{CD} \end{equation*}
is exact if and only if \(f\) is an isomorphism. Moreover,
\begin{equation*} \begin{CD} 0@>>>A@>f>>B@>g>>C@>>>0 \end{CD} \end{equation*}
is a short exact sequence if and only if - \(f\) is injective - \(g\) is surjective - \(\im f=\ker g\) So when this is indeed a short exact sequence, we can identify \(A\) with its image \(f(A)\text{,}\) which makes \(A = \ker g\text{.}\) Moreover, since \(g\) is surjective, by the First Isomorphism Theorem we conclude that \(C \cong B/A\text{,}\) so we might abuse notation and identify \(C\) with \(B/A\text{.}\) In particular, note that \(C = \coker f\text{.}\)
