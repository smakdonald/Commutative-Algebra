Proof

  1. (1) \(\Rightarrow\) (2): Let \(f_1, \ldots, f_n\) be homogeneous generators for \(I\text{.}\) If \(f\in I\text{,}\) we can write \(f\) as
    \begin{equation*} f = r_1 f_1 + \cdots + r_n f_n. \end{equation*}
    We can also write each \(r_i\) as a sum of its components, say \(r_i = [r_i]_{d_{i,1}} + \cdots + [r_i]_{d_{i,m_i}}\text{.}\) Then, after substituting and collecting,
    \begin{equation*} f= \sum_{d} ( [r_1]_{d-|f_1|} f_1 + \cdots + [r_n]_{d-|f_n|} f_n) \end{equation*}
    expresses \(f\) as a sum of homogeneous elements of different degrees, so
    \begin{equation*} f_d = [r_1]_{d-|f_1|} f_1 + \cdots + [r_n]_{d-|f_n|} f_n \in I. \end{equation*}
  2. (2) \(\Rightarrow\) (1): Any element of \(I\) is a sum of its homogeneous components. Thus, in this case, the set of homogeneous elements in \(I\) is a generating set for \(I\text{.}\)
  3. (1) \(\Rightarrow\) (3): As above, \(I\) is generated by the collection of additive subgroups \(\{I_a\}\) in this case; the sum is direct as there is no nontrivial \(\Z\)-linear combination of elements of different degrees.
  4. (3) \(\Rightarrow\) (2): We can take generators for each abelian group \(I \cap R_a\text{,}\) and the collection of all of them is a generating set for \(I\text{.}\)
