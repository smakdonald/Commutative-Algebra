Example 4.26. \(\ex\) -.

Fix a field \(k\text{,}\) and let \(R = k[x,y,z]\text{.}\) Consider the ideal \(P\) given by
\begin{equation*} P = \left( \underset{f}{\underbrace{x^3-yz}}, \underset{g}{\underbrace{y^2-xz}}, \underset{h}{\underbrace{z^2-x^2y}} \right). \end{equation*}
We will show that the ring homomorphism #empty is an isomorphism. First, note that it is immediately surjective by construction, so we just need to prove it is injective. If we set \(\deg(x)=3, \deg(y)=4, \deg(z)=5, \deg(t)=1\text{,}\) \(\pi\) is a graded homomorphism of graded rings, whose kernel is homogeneous. Since \([ k[t^3,t^4,t^5] ]_n\) is a \(1\)-dimensional vector space generated by \(t^n\) for all \(n\geqslant 3\) (and zero in degrees 1 and 2), it suffices to show that \(\dim( [\frac{k[x,y,z]}{P}]_n ) = 1\) for all \(n \geqslant 3\) (and zero in degrees 1 and 2).
Given any monomial in \(\frac{k[x,y,z]}{P}\text{,}\) we can use the relations \(y^2-xz\text{,}\) \(z^2-x^2y\text{,}\) and \(yz-x^3\) to obtain an equivalent monomial where the sum of the \(y\) and \(z\) exponents is smaller until we get a monomial of the form \(x^a\text{,}\) \(x^a y\text{,}\) or \(x^a z\text{.}\) If \(n=1,2\text{,}\) there is no such monomial; if \(n\geqslant 3\text{,}\) there is exactly one, namely,
\begin{equation*} \begin{cases} x^{n/3} & \text{if} \ n\equiv 0 \mod 3 \\ x^{(n-4)/3} y & \text{if} \ n\equiv 1 \mod 3 \\ x^{(n-5)/3} y & \text{if} \ n\equiv 2 \mod 3 \end{cases} \end{equation*}
This shows that \(P\) is the kernel of the map #empty Since \(k[t^3,t^4,t^5] \subseteq k[t]\) is a domain, we conclude that \(P\) is a prime ideal. In fact, \(P\) is a homogeneous ideal with the grading we considered above: our generators \(f\text{,}\) \(g\text{,}\) and \(h\) are now homogeneous, with \(\deg(f) = 9\text{,}\) \(\deg(g) = 8\text{,}\) and \(\deg(h) = 10\text{.}\)
We showed in () that \(\mathcal{Z}(P)\) is the variety
\begin{equation*} X = \{ (t^3, t^4, t^5) \mid t \in k \}. \end{equation*}
Thus we conclude that
\begin{equation*} \mathcal{I}(X) = P = \left( x^3-yz, y^2-xz, z^2-x^2y \right), \end{equation*}
and that \(k[t^3,t^4,t^5] \cong k[x,y,z]/P\) is the coordinate ring \(k[X]\text{.}\)
