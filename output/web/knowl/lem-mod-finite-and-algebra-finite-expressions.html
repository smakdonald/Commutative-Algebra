Lemma 1.28. Mod Finite and Algebra Finite Expressions.

If \(S\) is an \(R\)-algebra,
  1. \(R \subseteq S\) is algebra-finite if and only if \(S = R[f_1, \ldots, f_n]\) for some \(f_1, \ldots, f_n \in S\text{.}\)
  2. \(R \subseteq S\) is module-finite if and only if \(S = R f_1 + \cdots + R f_n\) for some \(f_1, \ldots, f_n \in S\text{.}\)
