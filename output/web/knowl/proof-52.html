Proof

Since \(k\) is a quotient of \(R\)[^2], the characteristic of \(R\) must be a multiple of the characteristic of \(k\text{,}\) since the map \(\Z\longrightarrow k\) factors through \(R\)[^3]. We must think of \(0\) as a multiple of any integer for this to make sense. Now \(k\) is a field, so its characteristic is \(0\) or \(p\) for a prime \(p\text{.}\)[^4] If \(\char(k)=0\text{,}\) then necessarily \(\char(R)=0\)[^5]. If \(\char(k)=p\text{,}\) we claim that \(\char(R)\) must be either \(0\) or a power of \(p\text{.}\) Indeed, if we write \(\char(R)=p^n \cdot a\) with \(a\) coprime to \(p\text{,}\) note that \(p\in \fm\text{,}\) so if \(a \in \fm\text{,}\) we have \(1\in (p,a)\subseteq \fm\text{,}\) which is a contradiction[^6]. Since \(R\) is local, this means that \(a\) is a unit.[^7] But then, \(p^n a=0\) implies \(p^n=0\text{,}\) so the characteristic must be \(p^n\text{.}\)
