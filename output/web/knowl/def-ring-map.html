Definition A.3. Ring Homomorphism.

Given rings \(R\) and \(S\text{,}\) a function \(R \xrightarrow{f} S\) is a ring homomorphism if \(f\) preserves the operations and the multiplicative identity, meaning
  • \(f(a+b) = f(a) + f(b)\) for all \(a, b \in R\text{,}\)
  • \(f(ab) = f(a)f(b)\) for all \(a, b \in R\text{,}\) and
  • \(f(1)=1\text{.}\)
A bijective ring homomorphism is an isomorphism. We should think about a ring isomorphism as a relabelling of the elements in our ring.
