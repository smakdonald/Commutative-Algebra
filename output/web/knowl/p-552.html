Let \(R=\C[x]\text{,}\) and \(M= \displaystyle\bigoplus_{n\in \Z} R/(x-n)\text{.}\) First, note that \(M_\fp = \displaystyle\bigoplus_{n\in \Z} (R/(x-n))_{\fp}\text{,}\) so
\begin{equation*} \Supp(M)= \bigcup_{n\in \Z} \Supp(R/(x-n))= \bigcup_{n\in \Z} V((x-n))=\{(x-n) \ | \ n \in \Z\}. \end{equation*}
On the other hand,
\begin{equation*} \ann_R(M)=\bigcap_{n\in \Z} \ann_R(R/(x-n)) = \bigcap_{n\in \Z} (x-n) = 0. \end{equation*}
Note that in this example the support is not even closed.
