Definition A.1. Ring.

A ring is a set \(R\) equipped with two binary operations \(+\) and \(\cdot\) satisfying the following properties:
  1. \(R\) is an abelian group under the addition operation \(+\text{,}\) with additive identity \(0\text{,}\) or \(0_R\) if we need to specify which ring we are talking about. Explicitly, this means that
    • \(a + (b+c) = (a+b) + c\) for all \(a, b, c \in R\text{,}\)
    • \(a + b = b + a\) for all \(a, b \in R\text{,}\)
    • there is an element \(0 \in R\) such that \(0 + a = a\) for all \(a \in R\text{,}\) and
    • for each \(a \in R\) there exists an element \(-a \in R\) such that \(a + (-a) = 0\text{.}\)
  2. \(R\) is a commutative monoid under the multiplication operation \(\cdot\text{,}\) with multiplicative identity \(1_R\) or simply \(1\text{.}\) Explicitly, this means that
    • \((a \cdot b) \cdot c = a \cdot (b \cdot c)\) for all \(a, b, c \in R\text{,}\)
    • \(a \cdot b = b \cdot a\) for all \(a, b \in R\text{,}\) and
    • there exists an element \(1 \in R\) such that \(1 \cdot a = a \cdot 1\) for all \(a \in R\text{.}\)
    We typically write \(ab\) for \(a \cdot b\text{.}\)
  3. multiplication is distributive with respect to addition, meaning that for all \(a, b, c \in R\) we have
    \begin{equation*} a \cdot (b+c) = a \cdot b+a \cdot c. \end{equation*}
  4. \(1 \neq 0\text{.}\)
