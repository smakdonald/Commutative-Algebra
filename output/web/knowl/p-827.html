Power series rings \(k \llbracket x_1, \ldots, x_d \rrbracket\) are regular local rings. In general, a ring is regular if all its localizations are regular local rings. In order for this definition to make sense, we need to first make sure that regularity localizes, meaning that if \((R, \fm)\) is a regular local ring, then \(R_P\) is also regular for all primes \(P\text{.}\) But to do that, we need some homological algebra. However (spoiler alert!), things do work out alright, and as you might expect, polynomial rings over fields are also regular.
