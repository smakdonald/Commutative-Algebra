Let \(W\) be multiplicatively closed in \(R\text{.}\) (a) If \(I\) is an ideal in \(R\text{,}\) then \(W^{-1} I = I W^{-1} R\text{.}\) (b) If \(I\) is an ideal in \(R\text{,}\) then \(W^{-1} I \cap R = \{ r \in R \ | \ wr\in I \textrm{ for some } w\in W\}\text{.}\) (c) If \(J\) is an ideal in \(W^{-1}R\text{,}\) then \(W^{-1}(J \cap R) = J\text{.}\) (d) If \(\fp\) is prime and \(W \cap \fp=\es\text{,}\) then \(W^{-1}\fp =\fp (W^{-1}R)\) is prime. (e) The map \(\Spec(W^{-1} R) \to \Spec(R)\) is injective, with image
\begin{equation*} \{\fp \in \Spec(R) \ | \ \fp \cap W = \varnothing\}. \end{equation*}
