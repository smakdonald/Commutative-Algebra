Item 1.

\(R\) is an abelian group under the addition operation \(+\text{,}\) with additive identity \(0\text{,}\) or \(0_R\) if we need to specify which ring we are talking about. Explicitly, this means that
  • \(a + (b+c) = (a+b) + c\) for all \(a, b, c \in R\text{,}\)
  • \(a + b = b + a\) for all \(a, b \in R\text{,}\)
  • there is an element \(0 \in R\) such that \(0 + a = a\) for all \(a \in R\text{,}\) and
  • for each \(a \in R\) there exists an element \(-a \in R\) such that \(a + (-a) = 0\text{.}\)
