We can also write the one-generated \(\Z\)-algebra \(\Z[\sqrt{d}]\) as a quotient of a polynomial ring in one variable: if \(d\) is not a perfect square, the map \(\pi\) in part c) of this [[Lemma - Generated Algebra Equivalencies|Lemma]] is
\begin{equation*} \begin{CD} \Z[x]@>\pi>> \Z[\sqrt[3]{d}]\\\\ x@>>> \sqrt[3]{d}\\\\ y@>>> \sqrt[3]{d^{2}} \end{CD} \end{equation*}
and its kernel is generated by \(x^2-d\text{,}\) so \(\Z[\sqrt{d}] \cong \Z[x]/(x^2-d)\text{.}\)
