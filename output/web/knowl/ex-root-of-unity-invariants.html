Example 1.88. Root of Unity Invariants.

Consider the group with two elements \(G=\{e,g\}\text{.}\) To define an action of \(G\) on \(R=k[x]\text{,}\) we need only to define \(g \cdot x\text{,}\) since \(e\) is the identity and \(g\) acts linearly. Consider the action of \(G\) on \(R=k[x]\) given by \(g \cdot x = -x\text{,}\) so \(g \cdot f(x) = f(-x)\text{.}\) Suppose that the characteristic of \(k\) is not 2, so \(-1 \neq 1\text{.}\) Write \(f= a_n x^n + a_{n-1} x^{n-1} + \cdots + a_0\text{.}\) We have \(g \cdot x^i=(-x)^i = (-1)^i x^i\text{,}\) so
\begin{equation*} g\cdot f = (-1)^n a_n x^n+ (-1)^{n-1} a_{n-1} x^{n-1} + \cdots + a_0, \end{equation*}
which differs from \(f\) unless for each odd \(i\text{,}\) \(a_i=0\text{.}\) That is, [ R^G = { f R  |  }.] Any such \(f\) is a polynomial in \(x^2\text{,}\) so we have
\begin{equation*} R^G = k[x^2]. \end{equation*}
In particular, \(R^G\) is a finitely generated \(k\)-algebra.
