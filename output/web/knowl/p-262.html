Let \(I\) be an ideal in a graded ring \(R\text{.}\) The following are equivalent:
  1. \(I\) is homogeneous.
  2. For any element \(f\in R\) we have \(f\in I\) if and only if every homogeneous component of \(f\) lies in \(I\text{.}\)
  3. \(I = \bigoplus_{a \in T} I_a\text{,}\) where \(I_a = I \cap R_a\text{.}\)
