Proposition 5.44. NAK 1.

Let \(R\) be a ring, \(I\) an ideal, and \(M\) a finitely generated \(R\)-module. If \(IM = M\text{,}\) then: (a) there is an element \(r\in 1 + I\) such that \(rM=0\text{,}\) and (b) there is an element \(a\in I\) such that \(am=m\) for all \(m\in M\text{.}\)
