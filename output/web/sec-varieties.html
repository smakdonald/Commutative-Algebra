    Colloquially, we often identify systems of equations with their solution sets. We will make this correspondence more precise for systems of polynomial equations, and develop the beginning of a rich dictionary between algebraic and geometric objects.

    Question 4.1.

    Let \(k\) be a field. To what extent is a system of polynomial equations
    \begin{equation*} \left\lbrace\begin{array}{c} f_1 = 0 \\ \vdots \\ f_t = 0 \end{array}\right. \end{equation*}
    with \(f_1, \ldots, f_t \in k[x_1, \ldots, x_d]\) determined by its solution set?

    Remark 4.2.

    Colloquially, we often identify systems of equations with their solution sets. We will make this correspondence more precise for systems of polynomial equations, and develop the beginning of a rich dictionary between algebraic and geometric objects.

    Question 4.3. Question -.

    Let \(k\) be a field. To what extent is a system of polynomial equations
    \begin{equation*} \left\lbrace\begin{array}{c} f_1 = 0 \\ \vdots \\ f_t = 0 \end{array}\right. \end{equation*}
    with \(f_1, \ldots, f_t \in k[x_1, \ldots, x_d]\) determined by its solution set?

    Definition 4.4. Affine \(n\)-Space.

    For an arbitrary field \(k\text{,}\) the affine \(n\)-space over \(k\text{,}\) written \(\A_k^n\text{,}\) is the set of \(n\)-tuples of elements of \(k\text{:}\)
    \begin{equation*} \A^n_k = \{(a_1,\dots,a_n) |a_i \in k\} \end{equation*}
    So, \(\A^n_k\) is just \(k^n\text{,}\) but we avoid using the latter since \(k^n\) is the usual notation of the standard \(n\)-dimensional \(k\)-vector space. Affine space is not thought of as a vector space typically.

    Definition 4.5. Zero Locus.

    For a subset \(T\) of \(k[x_1, \dots, x_d]\text{,}\) we define \(\mathcal{Z}(T) \subseteq \A^d_k\) to be the set of common zeros or the {} of the polynomials (equations) in \(T\text{:}\)
    \begin{equation*} \mathcal{Z}(T) := \{(a_1, \dots, a_d) \in \A^d_k \mid f(a_1, \dots, a_d) = 0 \text{ for all $f \in T$}\}. \end{equation*}
    A subset of \(\A^d_k\) of the form \(\mathcal{Z}(T)\) for some subset \(T\) is called an {} of \(\A^d_k\text{,}\) or an {}. A variety is {} if it cannot be written as the union of two proper subvarieties.

    Remark 4.6.

    Some authors use the word to refer only to irreducible algebraic sets. Note also that the definitions given here are only completely standard when \(k\) is algebraically closed.

    Remark 4.7.

    Note that if \(L \supseteq k\) are both fields, any polynomial \(f\in k[x_1,\dots,x_n]\) is also an element of \(L[x_1,\dots,x_n]\text{,}\) and we can evaluate it at any point in \(\A^n_L\text{.}\) Thus, we may write \(\mathcal{Z}_k(T)\) or \(\mathcal{Z}_L(T)\) to distinguish between the zero sets over different fields.

    Proposition 4.8. Properties of Z.

    Let \(k\) be a field and \(R=k[x_1, \dots,x_n]\text{.}\) Let \(S, T \subseteq R\) be arbitrary subsets. (a) If \(S \subseteq T\text{,}\) then \(\mathcal{Z}(S) \supseteq \mathcal{Z}(T)\text{.}\) (b) If \(I = (S)\) is the ideal generated by \(S\text{,}\) then \(\mathcal{Z}(S) = \mathcal{Z}(I)\text{.}\)

    Proof.

    First, note that imposing more equations can only lead to a smaller the solution set, which gives (1). For (2), we have \(\cZ(I) \subseteq \cZ(S)\) by (1). On the other hand, if \(f_1,\dots,f_m\in S\) and \(r_1,\dots,r_m\in R\text{,}\) and \((a_1,\dots,a_n)\in \cZ(S)\text{,}\) then \(f_i(a_1,\dots,a_n)=0\) for all \(i\text{.}\) Therefore,
    \begin{equation*} (\sum_i r_i f_i)(a_1,\dots,a_n) = \sum_i r_i(a_1,\dots,a_n) f_i(a_1,\dots,a_n) = 0, \end{equation*}
    so \((a_1,\dots,a_n)\in \cZ(\sum_i r_i f_i)\text{.}\) Thus, \((a_1,\dots,a_n)\in \cZ(I)\text{.}\) That is, \(\cZ(S) \subseteq \cZ(I)\text{.}\)*

    Example 4.9. \(\ex\) -.

    Let \(k\) be a field and \(R = k[x]\text{.}\) The varieties \(\cZ(x)\) and \(\cZ(x^2)\) are both the singleton given by the origin \(\{ 0 \}\) of \(\A^1_k\text{,}\) even though the ideals \((x)\) and \((x^2)\) are distinct.

    Proposition 4.10. Properties of Z Topology.

    Let \(k\) be a field and \(R=k[x_1, \dots,x_n]\text{.}\) Let \(I, I_\lambda, J\subseteq R\) be ideals in \(R\text{.}\) (a) \(\mathcal{Z}(R) = \mathcal{Z}(1) = \varnothing\) and \(\mathcal{Z}(0) = \A^n_k\text{.}\) (b) \(\bigcap_{\lambda \in \Lambda} \mathcal{Z}(I_\lambda) = \displaystyle \mathcal{Z} ( \sum_{\lambda \in \Lambda} I_\lambda)\text{.}\) (c) \(\displaystyle \cZ \left( I \cap J \right) = \cZ( IJ ) = \cZ(I) \cup \cZ(J)\text{.}\) ##### Proof. (1) is clear, since \(1\) is never equal to zero and \(0\) is always zero.
    Let‚Äôs now show (2). Given sets \(S_\lambda \subseteq T\text{,}\) a point is a solution to all of the equations in each set \(S_\lambda\) if and only if it is a solution of each set of equations \(S_\lambda\text{.}\) Therefore,
    \begin{equation*} \displaystyle \mathcal{Z} (\bigcup_{\lambda \in \Lambda} S_\lambda)=\bigcap_{\lambda \in \Lambda} \mathcal{Z}(S_\lambda). \end{equation*}
    Given ideals \(I_\lambda\text{,}\) \(\sum_{\lambda \in \Lambda} I_\lambda\) is the ideal generated by \(\bigcup_{\lambda\in\Lambda} I_\lambda\text{,}\) and thus [ ({} I) = ({} I) = {} (I).]
    To show (3), first consider subsets \(S, T \subseteq R\text{.}\) We claim that
    \begin{equation*} \cZ(S) \cup \cZ(T) \subseteq \cZ(\{fg \ | \ f\in S, g\in T\}). \end{equation*}
    Indeed, \(f(a_1,\dots,a_n)=0\) for all \(f\in S\) implies \(f(a_1,\dots,a_n) g(a_1,\dots,a_n)=0\) for all \(f\in S\) and all \(g\in T\text{.}\) On the other hand, if \((a_1,\dots,a_n)\notin \cZ(S) \cup \cZ(T)\text{,}\) then there is some \(f\in S\) and some \(g\in T\) with \(f(a_1,\dots,a_n) \neq 0\) and \(g(a_1,\dots,a_n)\neq 0\text{,}\) so \(f(a_1,\dots,a_n) g(a_1,\dots,a_n)\neq 0\text{.}\) Therefore,
    \begin{equation*} \cZ(S) \cup \cZ(T) \subseteq \cZ(\{fg \ | \ f\in S, g\in T\}). \end{equation*}
    Now given ideals \(I\) and \(J\text{,}\) since \(IJ \subseteq I \cap J \subseteq I\) and \(I \cap J \subseteq J\)), by (1) we get
    \begin{equation*} \cZ(I) \cup \cZ(J) \subseteq \cZ(I \cap J) \subseteq \cZ(IJ). \end{equation*}
    On the other hand, by (2) and (4) we get
    \begin{equation*} \cZ(IJ) \subseteq \cZ( \{ fg \ | \ f\in I , g\in J\}) = \cZ(I) \cup \cZ(J), \end{equation*}
    so the equalities hold throughout.

    Definition 4.11. Zariski Topology.

    Let \(k\) be a field. The collection of subvarieties \(X \subseteq \A^n_k\) is the collection of closed subsets in a topology on \(\A^n_k\text{.}\) This is called the on \(\A^n_k\text{.}\) Any subvariety of \(\A^n_k\) obtains a {} as the subspace topology from \(\A^n_k\text{.}\)

    Example 4.12. \(\ex\) -.

    Let \(k\) be an infinite field. The closed subsets in the Zariski topology on \(\A^1_k\) are just the finite subsets, along with the whole space. Note that this topology is not Hausdorff; quite on the contrary, any two nonempty open sets have infinite intersection!

    Definition 4.13. Vanishing Set.

    Given any subset \(X\) of \(\A^d_k\) for a field \(k\text{,}\) define
    \begin{equation*} \mathcal{I}(X) = \{g(x_1, \dots, x_d) \in k[x_1, \dots, x_d] \mid g(a_1, \dots, a_d) = 0 \text{ for all $(a_1, \dots, a_d) \in X$} \}. \end{equation*}

    Example 4.14. Exercise -.

    \(\mathcal{I}(X)\) is an ideal in \(k[x_1, \ldots, x_d]\) for any \(X \subseteq \A^d_k\text{.}\)

    Example 4.15. \(\ex\) -.

    For any field \(k\text{,}\) we have \(\mathcal{I}((a_1, \dots, a_d)) = (x_1 - a_1, \dots, x_d - a_d)\text{.}\)

    Remark 4.16.

    For a subset \(X\subseteq \A^n_K\) and a subset \(S \subseteq K[x_1,\dots,x_n]\text{,}\) we have
    \begin{equation*} X \subseteq \cZ(S) \Longleftrightarrow \text{each $s\in S$ vanishes at each $x\in X$} \Longleftrightarrow S \subseteq \cI(X). \end{equation*}

    Theorem 4.17. Properties of Vanishing Set.

    Let \(k\) be a field, and \(X,X_\lambda,Y\) be subsets of \(\A^n_k\text{.}\) (a) \(\cI(\varnothing) = R\) and, if \(k\) is infinite, \(\cI(\A^n_k) = 0\text{.}\) (b) If \(X \subseteq Y\text{,}\) then \(\cI(X) \supseteq \cI(Y)\text{.}\) (c) \(\cI(X)\) is a radical ideal.

    Proof.

    For (1), it is clear that \(\cI(\varnothing) = R\text{.}\) Now assume \(k\) is infinite. We show by induction on \(n\) that any nonzero polynomial in \(k[x_1,\dots,x_n]\) is nonzero at some point in \(\A^n_k\text{.}\) The case \(n=1\) is standard: a polynomial in \(k[x]\) of degree \(d\) can have at most \(d\) roots ‚Äî in particular, it cannot have infinitely many roots. Let \(n \geqslant 2\) and let \(f(x_1,\dots,x_n)\in k[x_1,\dots,x_n]\) be a nonzero polynomial. If \(f\) is a nonzero constant, it is nonzero at any point. Otherwise, we can assume that \(f\) nontrivally involves some variable, say \(x_n\text{.}\) Write
    \begin{equation*} f(x_1,\dots,x_n) = f_d(x_1,\dots,x_{n-1}) x_n^d + \cdots + f_0(x_1,\dots,x_{n-1}). \end{equation*}
    If \(f\) is identically zero, then for every \((a_1,\dots,a_{n-1})\in \A^{n-1}_k\text{,}\)
    \begin{equation*} f_d(a_1,\dots,a_{n-1}) x_n^d + \cdots + f_0(a_1,\dots,a_{n-1}) \end{equation*}
    is a polynomial in one variable that is identically zero, so it is the zero polynomial. Thus each \(f_i(a_1,\dots,a_{n-1})\) is identically zero. Since this holds for all \((a_1,\dots,a_{n-1})\in \A^{n-1}_k\text{,}\) by the induction hypothesis we conclude that each \(f_i\) is the zero polynomial. Therefore, \(f(x_1,\dots,x_n)\) is the zero polynomial, as required.
