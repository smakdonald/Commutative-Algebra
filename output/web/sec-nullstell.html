    Section 4.3 Nullstellensatz

    “If you are strong, appear weak. But if you are weak, appear strong.”
    ―Sun Tzu

    Lemma 4.27. Nullstellensatz Lemma 1.

    Let \(k\) be a field, and \(R=k[x_1,\dots,x_d]\) be a polynomial ring. There is a bijection #empty

    Proof.

    Each \(\fm = (x_1-a_1,\dots,x_d-a_d)\) is a maximal ideal satisfying \(R/\fm \cong k\text{.}\) Moreover, these ideals are distinct: if \(x_i-a_i,x_i-a'_i\) are in the same ideal for \(a_i\neq a'_i\text{,}\) then the unit \(a_i-a'_i\) is in the ideal, so it is not proper. Therefore, our map is injective. To see that it is surjective, let \(\fm\) be a maximal ideal with \(R/\fm \cong k\text{.}\) Each class in \(R/\fm\) corresponds to a unique \(a \in k\text{,}\) so in particular each \(x_i\) is in the class of a unique \(a_i \in k\text{.}\) This means that \(x_i - a_i \in \fm\text{,}\) and thus \((x_1-a_1,\dots, x_d-a_d)\subseteq \fm\text{.}\) Since \((x_1-a_i,\dots, x_d-a_i)\) is a maximal ideal, we must have \((x_1-a_1,\dots, x_d-a_d)= \fm\text{.}\)

    Example 4.28. \(\ex\) -.

    Not all maximal ideals in \(k[x_1, \ldots, x_d]\) are necessarily of this form. For example, if \(k = \mathbb{R}\) and \(d=1\text{,}\) the ideal \((x^2+1)\) is maximal, but
    \begin{equation*} k[x]/(x^2+1) \cong \mathbb{C} \not\cong k. \end{equation*}

    Corollary 4.29. Nullstellensatz.

    Let \(S=k[x_1,\dots,x_d]\) be a polynomial ring over an algebraically closed field \(k\text{.}\) There is a bijection \begin{center} #empty If \(R\) is a finitely generated \(k\)-algebra, we can write \(R=S/I\) for a polynomial ring \(S\text{,}\) and there is an induced bijection
    \begin{equation*} \mathcal{Z}_k(I)\subseteq \mathbb{A}^d_k \longleftrightarrow \{ \text{maximal ideals $\fm$ of $R$}\}. \end{equation*}

    Proof.

    The first part follows immediately from Lemma and . Since we skipped the proof of Zariski's Lemma, we give a proof of the case when \(k\) is an uncountable field.
    Let \(k\) be an uncountable algebraically closed field and \(\fm\) be any maximal ideal in \(R\text{.}\) Suppose that \(L := R/\fm \not\cong k\text{.}\) Then there exists some element \(\alpha \in L\) that is transcendental over \(k\text{,}\) since \(k\) is algebraically closed and \(L\) is an extension of \(k\text{.}\) Now consider the set
    \begin{equation*} S = \left\lbrace \frac{1}{\alpha-c} \mid c \in k \right\rbrace. \end{equation*}
    If \(S\) is a linearly dependent set over \(k\text{,}\) then we can find finitely many \(b_1, \ldots, b_n, c_1, \ldots, c_n \in k\) such that
    \begin{equation*} \frac{b_1}{\alpha-c_1} + \cdots + \frac{b_n}{\alpha-c_n} = 0. \end{equation*}
    Rewriting the left side as one fraction, the numerator is a polynomial in \(\alpha\) with coefficients in \(k\text{.}\) But \(\alpha\) is transcendental over \(k\text{,}\) so \(S\) is a linearly independent set. But \(k\) is uncountable, so \(S\) is uncountable, and that means that \(\dim_k(L)\) is uncountable. On the other hand, \(L\) is by construction algebra-finite over \(k\text{.}\) Since \(k\) is a field, being algebra-finite and module-finite over \(k\) are equivalent conditions, and thus \(L\) should be a finite dimensional vector space over~\(k\text{.}\) This is a contradiction, so we cannot have any transcendental element over \(k\) in \(L\text{.}\) Since \(k\) is algebraically closed, we conclude that \(k = L\text{.}\)
    To show the second statement, fix an ideal \(I\) in \(S\text{,}\) and \(R = S/I\text{.}\) The maximal ideal ideals in \(R\) are in bijection with the maximal ideals \(\fm\) in \(S\) that contain \(I\text{;}\) those are the ideals of the form \((x_1-a_1, \ldots, x_d-a_d)\) with \(I \subseteq (x_1-a_1, \ldots, x_d-a_d)\text{.}\) These are in bijection with the points \((a_1, \ldots, a_d) \in \mathbb{A}^d_k\) satisfying \((a_1, \ldots, a_d) \in \mathcal{Z}_k(I)\text{.}\)

    Theorem 4.30. Weak Nullstellensatz.

    Let \(k\) be an algebraically closed field. If \(I\) is a proper ideal in \(R=k[x_1,\dots,x_d]\text{,}\) then \(\mathcal{Z}_k(I)\neq\varnothing\text{.}\)

    Proof.

    If \(I\subseteq R\) is a proper ideal, then by there is a maximal ideal \(\fm \supseteq I\text{,}\) so \(\mathcal{Z}(\fm)\subseteq \mathcal{Z}(I)\text{.}\) Since \(\fm=(x_1-a_1,\dots,x_d-a_d)\) for some \(a_i \in k\text{,}\) \(\mathcal{Z}(\fm)\) is a point, and thus nonempty.

    Example 4.31. \(\ex\) -.

    Many ideals define the same solution set. For example, in \(R = k[x]\text{,}\) the ideals \(I_n = (x^n)\text{,}\) for any \(n \geqslant 1\text{,}\) all define the same solution set \(\mathcal{Z}_k(I_n) = \{ 0 \}\text{.}\)

    Example 4.32. Rabinowitz's Technique.

    Observe that, if \(f(\vx)\) is a polynomial and \(\underline{a} \in \mathbb{A}^d\text{,}\) \(f(\underline{a})\neq 0\) if and only if \(f(\underline{a}) \in k\) is invertible; equivalently, if there is a solution \(y=b\in k\) to \(yf(\underline{a})-1=0\text{.}\) In particular, a system of polynomial equations and inequations
    \begin{equation*} \left\lbrace \begin{array}{c} f_1(\vx)=0 \\ \vdots \\ f_m(\vx)=0 \end{array}\right. \textrm{ and } \left\lbrace\begin{array}{c} g_1(\vx)\neq 0 \\ \vdots \\ g_n(\vx)\neq 0 \end{array}\right. \end{equation*}
    has a solution \(\vx=\underline{a}\) if and only if the system
    \begin{equation*} \left\lbrace \begin{array}{c} f_1(\vx)=0 \\ \vdots \\ f_m(\vx)=0 \end{array}\right. \textrm{ and } \left\lbrace\begin{array}{c} y_1 g_1(\vx)-1=0 \\ \vdots \\ y_n g_n(\vx)-1=0 \end{array}\right. \end{equation*}
    has a solution \((\vx,\underline{y})=(\underline{a},\underline{b})\text{.}\) In fact, this is equivalent to a system in one extra variable:
    \begin{equation*} \left\lbrace \begin{array}{c} f_1(\vx)=0 \\ \vdots \\ f_m(\vx)=0 \\ y g_1(\vx)\cdots g_n(\vx)-1 = 0 \end{array}\right. \end{equation*}

    Theorem 4.33. Strong Nullstellensatz.

    Let \(R=k[x_1,\dots,x_d]\) be a polynomial ring over an algebraically closed field \(k\text{.}\) Let \(I\subseteq R\) be an ideal. The polynomial \(f\) vanishes on \(\mathcal{Z}_k(I)\) if and only if \(f^n\in I\) for some \(n \geqslant 1\text{.}\) In particular, \(\cI(\cZ(J)) = \sqrt{J}\text{.}\)

    Proof.

    Suppose that \(f^n\in I\text{.}\) For each \(\underline{a}\in \mathcal{Z}_k(I)\text{,}\) \(f(\underline{a})\in k\) satisfies \(f(\underline{a})^n =0 \in k\text{.}\) Since \(k\) is a field, \(f(\underline{a})=0\text{.}\) Thus, \(f\in \mathcal{Z}_k(I)\) as well.
    Suppose that \(f\) vanishes along \(\mathcal{Z}_k(I)\text{.}\) This means that given any solution \(\underline{a} \in \mathbb{A}^d\) to the system determined by \(I\text{,}\) \(f(\underline{a}) = 0\text{.}\) In other words, the system
    \begin{equation*} \left\lbrace \begin{array}{l} g(\vx)=0 \textrm{ for all } g \in I \\ f \neq 0 \end{array}\right. \end{equation*}
    has no solutions. By the discussion above, \(\mathcal{Z}_k(I+(yf-1))=\varnothing\) in a polynomial ring in one more variable. By the Weak Nullstellensatz, we have \(I R[y]+(yf-1) = R[y]\text{,}\) and equivalently \(1 \in I R[y]+(yf-1)\text{.}\) Write \(I=(g_1(\vx),\dots,g_m(\vx))\text{,}\) and
        \begin{center}
        $1 = r_0(\vx,y) (1-yf(\vx))+r_1(\vx,y) g_1(\vx) + \cdots + r_m(\vx,y) g_m(\vx).$
        \end{center}
    We can map \(y\) to \(1/f\) to get
        \begin{center}
        $1 = r_1(\vx,1/f) g_1(\vx) + \cdots + r_m(\vx,1/f) g_m(\vx)$
    \end{center}
    in the fraction field of \(R[y]\text{.}\) Since each \(r_i\) is polynomial, there is a largest negative power of \(f\) occurring; say that \(f^n\) serves as a common denominator. We can multiply by \(f^n\) to obtain \(f^n\) as a polynomial combination of the \(g\)'s.

    Lemma 4.36. Irreducible Variety and Prime Vanishing Set.

    A variety \(X \subseteq \A^d_k\) is irreducible if and only if \(\mathcal{I}(X)\) is prime.

    Proof.

    Suppose that \(X\) is reducible, say \(X = V_1 \cup V_2\) for two varieties \(V_1\) and \(V_2\) such that \(V_1, V_2 \subsetneq X\text{.}\) Note that this implies that \(\mathcal{I}(X) \subsetneq \mathcal{I}(V_1)\text{,}\) \(\mathcal{I}(X) \subsetneq \mathcal{I}(V_2)\text{,}\) and \(\mathcal{I}(X) = \mathcal{I}(V_1) \cap \mathcal{I}(V_2)\text{.}\) Then we can find \(f \in \mathcal{I}(V_1)\) such that \(f \notin \mathcal{I}(V_2)\text{,}\) and \(g \in \mathcal{I}(V_2)\) such that \(g \notin \mathcal{I}(V_1)\text{.}\) Notice that by construction \(fg \in \mathcal{I}(V_1) \cap \mathcal{I}(V_2) \mathcal{I}(X)\text{,}\) while \(f \notin \mathcal{I}(X)\) and \(g \notin \mathcal{I}(X)\text{.}\) Therefore, \(\mathcal{I}(X)\) is not prime.
    Now assume that \(\mathcal{I}(X)\) is not prime, and fix \(f, g \notin \mathcal{I}(X)\) with \(fg \in \mathcal{I}(X)\text{.}\) Then
    \begin{equation*} X \subseteq \mathcal{Z}(fg) = \mathcal{Z}(f) \cup \mathcal{Z}(g). \end{equation*}
    The intersections
    \begin{equation*} V_f = \mathcal{Z}(f) \cap X = \mathcal{Z}(\mathcal{I}(X) + (f)) \quad \textrm{ and } \quad V_g = \mathcal{Z}(g) \cap X = \mathcal{Z}(\mathcal{I}(X) + (g)) \end{equation*}
    are varieties, and \(X = V_f \cup V_g\text{.}\) Finally, since \(f \notin \mathcal{I}(X)\text{,}\) then \(X \not\subseteq V_f\text{.}\) Similarly, \(X \not\subseteq V_g\text{.}\) Thus \(X\) is reducible.

    Example 4.37. \(\ex\) -.

    In \(k[x,y,z]\text{,}\) the radical ideal \(I = (xy,xz,yz)\) corresponds to the variety \(X\) given by the union of the three coordinate axes. Each of these axes is a variety in its own right, corresponding to the ideals \((x,y)\text{,}\) \((x,z)\) and \((y,z)\text{.}\) The three axes are the irreducible components of \(X\text{.}\) And indeed, \((x,y)\text{,}\) \((x,z)\) and \((y,z)\) are the three minimal primes over \(I\text{,}\) and
    \begin{equation*} (xy,xz,yz) = (x,y) \cap (x,z) \cap (y,z). \end{equation*}
